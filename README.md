# Shrinky-Image

Shrinky-Image es una aplicación web moderna diseñada para optimizar, comprimir y convertir imágenes directamente desde tu navegador. Con procesamiento local gracias a WebAssembly y un diseño responsivo impulsado por Tailwind CSS, esta herramienta garantiza eficiencia, privacidad y resultados de alta calidad.

## Características principales
- Compresión avanzada con ajustes personalizados.
- Conversión entre formatos (JPEG, PNG, WebP, AVIF).
- Redimensionamiento dinámico según tus necesidades.
- Comparación visual antes y después de la optimización.
- Procesamiento local que asegura la privacidad.

## Tecnologías utilizadas
- **Next.js**
- **Tailwind CSS**
- **WebAssembly**

## Instalación y uso
1. Clona este repositorio.
2. Instala las dependencias con `npm install`.
3. Inicia el servidor de desarrollo con `npm run dev`.
